# FIWARE

## Descripción general

Este contenido esta diseñado para aprender las funciones de los componentes de FIWARE en formato de taller práctico.

Se centra en las funciones de FIWARE Orion, incluyendo la creación de datos históricos, la recopilación de datos desde dispositivos IoT, la protección de acceso a los componentes de FIWARE y la visualización de datos.


## Cómo usarlo

1. Disponer de un entorno donde puedas usar docker, docker compose, curl, git, etc.
2. Clonar este repositorio en el entorno anterior.
